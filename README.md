# Typescript & SystemJs & RxJs Training Project

- This project built with Typescript, SystemJs, RxJs

## Setup and Run Project

```bash
$ git clone git@github.com:euclid1990/typescript-systemjs.git my-proj
$ cd my-proj
$ npm install
$ bower install
$ gulp default
$ npm start
```
